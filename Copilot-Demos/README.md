4. **Show before/after** code transformations

## Tips for Effective Demos

### Before the Demo
- Ensure Copilot extension is active (check status bar)
- Pre-open relevant files
- Have Azure CLI logged in for validation

### During the Demo
- Speak prompts aloud before typing
- Pause to show Copilot suggestions
- Accept suggestions with Tab, reject with Escape
- Use Copilot Chat for explanations

### Key Features to Highlight
- **Inline completions** - Type and see suggestions
- **Chat interface** - Natural language interaction
- **Slash commands** - `/explain`, `/fix`, `/tests`
- **@workspace** - Context-aware responses
- **gh copilot** - CLI integration
